# courseforge:brainstorm

Captures ideas from the user in a structured way and saves them to `draft/ideas.md`. Can be invoked at any point without interrupting the current step — brainstorming is additive, not a detour.

---

## What to do

### Step 1 — Open the session

Tell the user this is a free space to capture ideas — examples, topics, activities, angles, anything that comes to mind for the course. Nothing gets committed to the course yet; it all goes into `draft/ideas.md` for later use.

Invite them to share whatever they have. Don't structure the prompt too tightly — the goal is to surface things, not filter them.

---

### Step 2 — Listen and reflect

As the user shares ideas, engage with them briefly:
- Ask a quick follow-up if an idea is too vague to remember later ("What would that exercise look like?")
- Note connections between ideas if you see them ("That could work well in Module 2 when you cover X")
- Don't evaluate or reject ideas during brainstorming — capture everything

---

### Step 3 — Organize before saving

Before saving, group the ideas into loose categories if there are enough to warrant it. Common groupings:
- Content ideas (topics, angles, examples)
- Activity ideas (exercises, projects, challenges)
- Design ideas (visual approaches, formats, structures)
- Course-level ideas (structure, sequencing, scope)

If there are only a few ideas, a flat list is fine.

---

### Step 4 — Save to ideas.md

Read `draft/ideas.md` first, then append the new ideas without overwriting what's already there.

Format each idea as a short bullet. Add a category label if grouping was used:

```markdown
## [Category]
- [Idea]
- [Idea]
```

If no categories, just append bullets under a `---` divider with today's date.

---

### Step 5 — Confirm and return

Tell the user how many ideas were saved and confirm they're in `draft/ideas.md`. 

Then ask: do they want to keep brainstorming, or return to where they were? If they want to return to the previous step, check `draft/progress.md` and invoke the appropriate skill to resume.
