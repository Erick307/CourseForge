# Course Creation Workflow

---

## Phase 1 — Structure

### Define the Course

Establish the three constraints that shape everything else. Iterate between them until they form a coherent, realistic definition.

- **Audience** — Who is this for? What do they know coming in, and why are they taking this course?
- **Goal** — What will learners be able to do by the end?
- **Length** — How much time does the learner have? Adjust goal or scope to fit.

**Output:** A brief course definition (2–4 sentences) that answers: who, what outcome, and how long.

### Define Scope, Prerequisites, and Structure

Deepen the course definition into a concrete blueprint.

- **Scope** — What is explicitly included? What is explicitly *out of scope*? Naming both prevents drift.
- **Prerequisites** — What knowledge must the learner already have before starting?
- **Module structure** — List each module with a one-line description and its order.

**Output:** A structured outline listing scope boundaries, prerequisite knowledge, and an ordered module list with descriptions.

### Define Activities

For each module, design the activities that learners will complete. Content will later be built around these.

- **Activity description** — What will the learner do?
- **Alignment** — Which module goal does this activity serve?
- **Quantity** — One or more activities per module depending on complexity; some modules may have none.

**Output:** An activity list mapped to modules, each with a description and its associated module goal.

---

## Phase 2 — Design

### Define Branding

Establish the visual identity that will be applied consistently across all course materials.

- **Colors** — Primary and secondary palette.
- **Fonts** — Heading and body typefaces.
- **Logos** — Any brand marks or course-specific assets.

**Output:** A branding reference (colors, fonts, logos) to be used in all subsequent design steps.

### Define Slide Structure and Define Content Structure *(any order)*

Both depend on branding but are independent of each other.

#### Define Slide Structure
Define the slide types and layout rules for this course.

**Output:** A presentation template with the reusable slide layouts for this course.

#### Define Content Structure
Define the structure and layout of the written course documents.

**Output:** A document template that all course content will follow.

---

## Phase 3 — Production

Repeat for each module in order:

#### Write the Content Document
Using the document template, write the full content document for the module.

**Output:** A completed content document for the module.

#### Develop the Activity
Using the activity definition, write the full activity for the module.

**Output:** A completed, ready-to-use activity for the module.

#### Build the Slides
Using the presentation template and the content document, create the slides for the module.

**Output:** A completed slide deck for the module.
